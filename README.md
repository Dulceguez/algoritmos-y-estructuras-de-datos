# Algoritmos y Estructuras de Datos

Repositorio con ejercicios y trabajos prácticos de la materia **Algoritmos y Estructuras de Datos** (2° año), cuyo objetivo es enseñar a diseñar, analizar y programar algoritmos eficientes usando estructuras de datos fundamentales.

---

## 📚 Contenidos principales

- Estructuras de datos no lineales: listas, árboles y grafos  
- Recursión y su aplicación en algoritmos  
- Árboles binarios y generales: construcción, recorridos (InOrden, PreOrden, PostOrden), búsqueda, inserción y borrado  
- Colas de prioridad y heaps: operaciones y ordenamiento Heapsort  
- Análisis de algoritmos: complejidad, notación O(), Θ, Ω; mejor, promedio y peor caso  
- Grafos: representaciones, recorridos DFS/BFS, componentes conexas, caminos mínimos  
- Algoritmos clásicos: Dijkstra, Floyd, Prim y Kruskal  
- Ordenamiento topológico y sus aplicaciones

> Los contenidos se desarrollan mediante ejercicios prácticos, implementación en Java y resolución de problemas.
